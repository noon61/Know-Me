# 主な機能
<!-- 作成したプロダクトを機能ごとに説明 -->
<!-- スクリーンショットやGIF，プロモーション画像等を貼る -->

- 機能1:ユーザ情報登録機能
- 名前、学年、研究室、サークル、履修科目等を登録できる。

- 機能2:学生検索機能
- 研究室、サークル、履修科目から学生を検索できる。

- 機能3: メッセージ
- 検索した学生にメッセージを送り、知りたい情報を聞くことができる。
